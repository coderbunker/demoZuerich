## Usage

After pulling from git, you can run the code with:

```
react-native run
```

or for android:

```
react-native run-android
```

or for ios:

```
react-native run-ios
```

To see log output use for android:

```
react-native log-android
```

And for ios:

```
react-native log-ios
```
